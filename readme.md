# Personalized News Summarizer

An end-to-end Machine Learning project that:

- Fetches news from APIs and RSS feeds
- Classifies articles into categories
- Generates summaries using NLP
- Personalizes recommendations based on user interests
- Evaluates performance using ROUGE and ranking metrics

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- Transformers (BART)
- SQLite
- Streamlit
- Git & GitHub
